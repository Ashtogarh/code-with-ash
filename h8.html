UNORDERED LISTS

EXPLANATION: The HTML < UL>tag defines an unordered list.The list items will be marked with bullets.Unordered List is a type of list where the data or elements can be arranged in the list in no specific order. An unordered list indicates the unordered list type. An unordered list starts with the < UL> tag and ends with < /UL>tag. The list items will be marked with bullets.

EXAMPLE:

FRUITS
< Ul>
< li> APPLE< /li>
< li>BANNANA
< li>MANGO
< /Ul>

OUTPUT:
FRUITS:

  • APPLE
  • BANNANA
  • MANGO

THE STYLE TYPE PROPERTY

CSS/style property controls the type of bullet to be used in this tag.

PROPERTY-LIST-STYLE-TYPE

There can be different types of bulleted list:
1. list-style-type: circle(will ive solid black circle )
2.list-style-type:square (will give a solid black square )
3. list-style-type:disc(will give bulets)
